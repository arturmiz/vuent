# Tabs

[[toc]]

## Basic

<tabs-basic />

```html
<vnt-tabs>
  <vnt-tab label="Tab 1" active>
    <p>...</p>
  </vnt-tab>
  <vnt-tab label="Tab 2">
    <p>...</p>
  </vnt-tab>
  <vnt-tab label="Tab 3">
    <p>...</p>
  </vnt-tab>
</vnt-tabs>
```

## Configuration

#### `active` attribute

Determines which tab should be set as active.
Must be added to `vnt-tab` element that should be active at start.
